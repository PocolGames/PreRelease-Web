# 첫 번째 포스팅 🚀

**PreRelease**에 오신 것을 환영합니다!

이곳은 특별한 순간들을 미리 공개하는 공간입니다.

## 주요 기능
- **Markdown 지원**: 텍스트 포맷팅이 가능합니다
- **이미지 갤러리**: 여러 장의 사진을 한 번에 업로드
- **반응형 디자인**: 모든 기기에서 완벽하게 작동

> 이것은 인용구입니다. Twitter 스타일의 블랙 테마로 제작되었어요!

```javascript
// 코드 블록도 지원합니다
console.log("Hello, PreRelease!");
```
